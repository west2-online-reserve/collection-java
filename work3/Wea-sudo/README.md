# 第三次作业 - 订单管理系统

## 作业简介
本次作业实现了一个基于Java + JDBC的简易订单管理系统，核心用于记录商品、订单相关信息并持久化到MySQL数据库。系统涵盖商品增删改查、订单创建/删除/查询、库存联动、事务管理等核心功能，同时通过分层设计（实体层、DAO层、服务层、工具类）保证代码的可读性与可维护性，通过数据校验、SQL预编译、逻辑删除等方式保障数据安全与完整性。

### 核心功能
1. 商品管理：支持商品新增、补货、查询、逻辑删除，包含价格/库存合法性校验；
2. 订单管理：支持订单创建（关联商品扣减库存）、订单删除（级联处理订单项）、订单/订单项查询；
3. 数据持久化：基于JDBC实现与MySQL的交互，封装JDBC工具类统一管理数据库连接、资源释放、事务控制；
4. 数据校验：创建订单时校验商品存在性、库存充足性、价格/数量合法性，保证订单信息完整准确。
