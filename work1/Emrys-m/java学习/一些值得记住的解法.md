# 一些值得记住的解法

1. **秦九韶算法**

   对于多项式 ![截图](file:///C:\Users\20240715\AppData\Local\Temp\ksohtml19264\wps2.jpg)，可以将其重写为：
   *f*(*x*)=*a*[0]+*x*×(*a*[1]+*x*×(*a*[2]+⋯+*x*×(*a*[*n*−1]+*x*×*a*[*n*])⋯))，

   具体实现：

   ```C
   double f( int n, double a[], double x ){
       double sum = a[n];
       for(int i = n - 1 ; i >= 0 ; i--){
           sum = sum * x + a[i];
       }
       return sum;
   }
   ```
